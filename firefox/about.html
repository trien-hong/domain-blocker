You can virtually add any domain to a list. When you try to visit the website it'll redirect you to 1 of 10 random extension page.

That is assuming you have added the domain to the list. To block a domain, go to the extension options page and add the website there.

Sometimes, ad blockers don't block everything on a website. By manually adding the domain of the ad you should be able to block it.

Common Errors

It's possible to add an incorrect domain. The incorrect domain just won't be blocked by the extension.

The correct URL only consist of any number of subdomains + domain + top level domain. IP address will also work.

Try to avoid special characters if possible (#`^{}|"<? etc.).

Try to avoid paths if possible (/aboutus)

Do not included scheme (http/https).

Do not included leading special characters (://).

Do not included spaces (" ").

There are some cases where it will also not work.
